# MateCheck - 제품 요구사항 문서 (PRD)

## 1. 프로젝트 개요
**프로젝트명:** MateCheck  
**목표:** 대가족, 맞벌이 부부, 신혼부부, 동거 연인, 룸메이트 등 "보금자리(Nest)"를 공유하는 구성원들이 일상 생활의 목표, 일정, 할 일을 공유하고 소통할 수 있는 모바일 앱.  
**핵심 가치:** "함께 가꾸는 우리집 관리자" - 딱딱한 업무 관리가 아닌, 아기자기하고 시각적인 즐거움이 있는 생활밀착형 공유 플랫폼.

## 2. 주요 타겟 유저
- **가족:** 일정 공유 및 가사 분담이 필요한 맞벌이 부부 및 대가족
- **커플:** 데이트 일정, 공동 목표 관리가 필요한 신혼부부 및 동거 연인
- **룸메이트:** 청소 당번, 공금 관리 등이 필요한 하우스메이트

## 3. 핵심 기능 명세

### 3.1 회원가입 및 온보딩 (보금자리 생성)
- **간편 회원가입:** 소셜 로그인 등을 통한 빠른 진입.
- **프로필 설정:**
  - `보금자리` 내에서 사용할 닉네임 설정.
  - 아바타 선택 (캐릭터 커스터마이징 또는 프리셋 선택).
- **보금자리(Nest) 생성/참여:**
  - **다중 보금자리 지원:** 가족, 회사 동료, 동아리, 룸메이트 등 여러 그룹(보금자리)을 생성하고 자유롭게 전환하며 사용 가능.
  - **생성:** 보금자리 이름 짓기, 대표 이미지(배경) 선택.
  - **참여:** 초대 코드/링크를 전달받아 기존 보금자리에 입장.
- **다양한 구성원 지원:** 사람 외에 반려동물, 식물, 아기, AI 에이전트 등을 구성원으로 추가하여 관리 가능.
- **AI 행복 매니저:** 
  - 단순 알림이 아닌, "가족의 행복을 위해 챙겨주는" 친절하고 따뜻한 어조의 안내 멘트 제공.
  - 일정을 빠뜨리지 않게 선제적으로 챙겨주는 집사 역할.

### 3.2 메인 홈 (보금자리 시각화)
- **Nest View:** 설정한 배경과 보금자리 이름이 표시됨.
- **아바타 배치:** 현재 보금자리에 속한 구성원(유저, 반려동물 등)의 아바타가 공간에 배치되어 "함께 있다"는 느낌 강조.
- **상태 메시지/가훈:** 화면 상단 또는 중앙에 "가훈"이나 "공동 목표"를 텍스트로 띄울 수 있음.

### 3.3 공동 투두리스트 (미션 & 체크)
- **작업 관리:** 해야 할 일(Mission) 생성, 수정, 삭제.
- **반복 설정:** 매일/매주/매월 등 반복 루틴 설정 (예: 개 산책, 화초 물주기, 약 먹기).
- **완료 체크:** 
  - 구성원 중 누구라도 완료 체크 가능.
  - 완료 시 "누가" 했는지 시각적으로 표시 (아바타 활용 등).
  - 완료 이력 로그 확인.

### 3.4 공동 캘린더
- **일정 공유:** 심플하고 깔끔한 UI의 월간/주간 캘린더.
- **일정 타입:** 개인 일정(공유 옵션), 공동 일정.
- **투표 기능:** 약속 날짜 잡기 등을 위한 투표 기능 (예: "이번 주 외식 언제 할까?" -> 투표 마감일 설정).

### 3.5 시스템 및 기타
- **알림(Push):** 일정 임박, 투두 미완료 알림, 투표 독려 등.
- **튜토리얼:** 앱 최초 진입 시 주요 기능을 시연하는 인터랙티브 튜토리얼 제공.

### 3.6 수익 모델 (Premium) & 심화 기능
- **구독 모델:** 기본 기능 무료, 심화 기능 유료.
- **가족 가계부 (공동 자금 관리 - Premium):**
  - **목표 설정:** 공동 자금 목표 세우기 (예: 여행 자금, 내 집 마련).
  - **지출 관리:** 매일매일의 지출 현황 기입 및 공유.
  - **예산 및 청구 관리:** 
    - 관리비/공과금 등의 '예상 금액'을 미리 설정(Pre-fill).
    - 실제 고지서 청구 후 '확정 금액'으로 수정 및 납부 체크 기능.
  - **유료 기능 예시:** 심화 가계부 분석 리포트, 프리미엄 AI 코칭, 등.

## 4. 비기능적 요구사항 (Non-functional)
- **톤앤매너(Tone & Manner):**
  - **AI 페르소나:** 딱딱한 시스템 메시지가 아닌, "오늘 비가 오는데 화분에 물 주는 거 잊지 않으셨죠?" 와 같이 말을 건네는 따뜻한 말투.
- **디자인(Aesthetics):** 
  - 풍부한 색감(Vibrant Colors), 다이내믹한 애니메이션.
  - "보금자리"라는 따뜻하고 감성적인 느낌을 주면서도 세련된 UI.
  - 반응형 마이크로 인터랙션 (버튼 클릭, 완료 체크 시의 쾌감 등).
- **플랫폼:** React Native (Expo) 기반 iOS/Android 앱.

## 5. 데이터 구조 (간략)
- **User:** ID, Nickname, AvatarID, CurrentNestID
- **Nest:** ID, Name, ImageURL, Motto, Members(List<User/Entity>)
- **Todo:** ID, Title, Recurrence, AssignedTo, CompletedBy, Date
- **Event:** ID, Title, Date, VoteData
