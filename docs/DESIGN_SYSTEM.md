# MateCheck - Design System

## 1. Design Philosophy
- **Keywords:** Cozy, Vibrant, Connected, Lively.
- **Core Value:** "Make the screen feel like a home."
- **Interaction:** Smooth, bouncy micro-animations to reward user actions.

## 2. Voice & Tone (AI Persona)
- **Role:** The "Happy Home Manager" (가족의 행복을 챙기는 집사).
- **Tone:** Kind, Warm, Proactive, Encouraging (친절함, 따뜻함, 선제적, 독려하는).
- **Writing Style:**
  - Avoid: "Task Completed.", "Deadline approaching." (Robotic)
  - Preferred: "Great job! Another happy moment recorded.", "Just a gentle reminder, tomorrow is trash day!" (Conversational)
- **Localization:** Korean (Honorifics: '해요'체, polite but friendly).

## 3. Color Palette (Proposed)
*These colors aim to be modern yet warm.*

### Primary (Brand)
- **Primary Main:** `#FF7F50` (Coral) - 따뜻하고 활기찬 느낌.
- **Primary Light:** `#FFAB91` - 부드러운 강조.
- **Primary Dark:** `#E64A19` - 중요 버튼/액션.

### Secondary (Accent)
- **Secondary:** `#4DB6AC` (Soft Teal) - 안정감, 힐링(식물/펫).
- **Vote/Active:** `#5C6BC0` (Indigo) - 신뢰, 캘린더 일정.

### Background & Surface
- **Background:** `#F9FAFB` (Warm Gray 50) - 눈이 편안한 배경.
- **Paper/Card:** `#FFFFFF` (White) with clean shadows.
- **Text Main:** `#374151` (Gray 700)
- **Text Sub:** `#9CA3AF` (Gray 400)

## 4. Typography
*System Font (San Francisco on iOS, Roboto on Android) or Custom Font (Pretendard/Inter).*
- **Heading 1 (Nest Name):** Size 28px, Bold, Rounded.
- **Heading 2 (Section Title):** Size 20px, SemiBold.
- **Body:** Size 16px, Regular.
- **Micro:** Size 12px, Medium (for tags, dates).

## 5. UI Components Guidelines

### Avatars
- **Shape:** Circle (default) or Squircle (modern).
- **Status:** Should have a ring or indicator when task is done or active.

### Cards (Clean & Modern)
- **Style:** Opaque white background or subtle gradients.
- **Border:** None or very subtle stroke.
- **Shadow:** Soft ecosystem shadows.

### Buttons
- **Primary:** Pill-shaped (fully rounded corners), Gradient background optional.
- **Secondary:** Outline or ghost style.

### Micro-Interactions
- **Check Task:** Scale up and down (bouncing effect), particles/confetti.
- **Navigation:** Smooth fade or slide transitions.
